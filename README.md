# Control de Asistencia (Angular)

Aplicación desarrollada en Angular para gestionar y controlar la asistencia.

## Tecnologías
- Angular
- TypeScript

## Funcionalidades (resumen)
- Gestión de módulos/pantallas para asistencia
- Componentes y rutas para navegación
- Estructura lista para escalar

## Requisitos
- Node.js 18+

## Instalación
```bash
npm install

Ejecutar
npm start

Build
npm run build
